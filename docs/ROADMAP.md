# Project Roadmap

## Backlog

Ideas and feature requests for future consideration.

### Browser-Accessible Downloads
**Domain:** WebRTC Mode
- **Feature:** Allow a browser-based receiver to pick up a transfer via the same
  Nostr signaling and WebRTC data channel, eliminating the need to install the
  `beam-rs-webrtc` CLI on the receiving machine.
- **Benefit:** Lower friction for one-off receivers who already have a browser.
